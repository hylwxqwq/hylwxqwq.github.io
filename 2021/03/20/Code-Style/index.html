Code Style

发表于 2021-03-20

RT。

时间：20/3/8

  1. 万能头
  2. 快读板子封装
  3. 代码块和代码块，空行区别。
  4. 大括号不换行。
  5. for里register
  6. while 和 if 如果后面能只用一行就不打括号。(if 后是return 除外
  7. else if 打括号。
  8. 线段树Define来写。
  9. 写图论喜欢写 u，v
  10. ++i 和 i++ 随缘
  11. 一道题如果要多个数据结构，会用namespace封装。
  12. 如果不是撞关键字绝对不大写。
  13. 讨厌空格。
  14. 能连等直接连着写。
  15. 同类型（功能）操作一般咋子一行用逗号隔开一下。
  16. 尖括号能不空格就不空。
  17. maxn不写，写si（size缩写）
  18. 一个模块的变量定义到一起。
