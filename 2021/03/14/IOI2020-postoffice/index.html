[IOI2000] 邮局

一道非常有意思的题目。

description

高速公路旁边有一些村庄。高速公路表示为整数轴，每个村庄的位置用单个整数坐标标识。没有两个在同样地方的村庄。两个位置之间的距离是其整数坐标差的绝对值。

邮局将建在某些村庄。为了建立邮局，应选择他们建造的位置，使每个村庄与其最近的邮局之间的距离总和最小。

你要编写一个程序，已知村庄的位置和邮局的数量，计算每个村庄和最近的邮局之间所有距离的最小可能的总和。

1<=村庄数<=3000，1<=邮局数<=3000,1<=村庄坐标<=10000

输入

第一行包含两个整数：第一个是村庄的数量n，第二个是邮局的数量m

第二行包含n个整数。这些整数是村庄的位置。

输出

一个整数S，它是每个村庄与其最近的邮局之间的所有距离的总和。

soltion

我们根据题目，不难想到：

设 $f[i][j]$ 表示考虑前 $i$ 个村, 放置 $j$ 个邮局的最小值。

这里有我和机房同学的一些错误想法：link

那么我们可以想到，村庄的位置是固定的。

但是邮局位置不固定。

所以我们的重心就要放在邮局的位置枚举上。

第一想法就是这样的：

for(register int i=1;i<=n;++i){
	for(register int j=1;j<=m;++j){
		f[i][j]=min(f[i][j]+min(f[i-1][j]+cost,f[i-1][j-1]));
	}
}

但是很明显不行，因为这样子没有考虑到我们这个决策会更新到前面的决策的情况。（也就是有后效性了）。

既然如此，我们死就死在邮局的位置决策的局限性（这种做法断定决策时邮局不是在前一个就是在这一个，而且无法更新全部的贡献值）。

那么我们就再搞一个循环/状态/函数，去枚举这个邮局出现的位置。

很明显，邮局可以出现在我们枚举到的 $i$ 位置之前的所有地方。

所以我们再加一层循环。

枚举上一个邮局出现的地方。

for(register int i=1;i<=n;++i){
	for(register int j=1;j<=m;++j){
		for(register int k=0;k<i;++k){ //这里为什么是 [0,i),等会会说。
           //决策
		}
	}
}

考虑里面怎么转移。

我们考虑，我们在枚举的 $k$ 位置放了一个邮局。

那么我们的状态就要从 $f[k][j-1]$ 过来。

那么，这个时候这里出现了一个邮局，它附近的村庄的贡献值就要更新。

考虑更新。

我们在枚举的时候，只考虑了 $[k,i]$ 所以不要去管之后的状态

那么很明显之有 $[k,i]$ 会被影响。

所以我们只需要计算这个区间的贡献就好了。

之所以我们要从 $0$ 开始，是为了更新初值。

然后我们知道，$0$ 位置没有村庄，所以我们考虑 $[k+1,i]$ 区间的贡献就好了。

然后我们这里加了一个 $1$ ，所以直接枚举 $[k+1,i)$

然后你谷题解去有大佬已经说了：

若村庄数为奇数，放中位数处距离和最小。若村庄为偶数，放中间两个村庄之间任意一处均可。

然后根据它计算贡献就好了。

Code：

#include<bits/stdc++.h>
using namespace std;

const int si=300;
int n,m;
int f[si][si];//f[i][j] means ,we put j villages in [1,i] ,the minvalue we can get.
int dist[si];//distance of each village.

int calc(int l,int r){
	int rt=0;
	int mid=(l+r)>>1;
	for(register int i=l;i<=mid;++i){
		rt+=dist[mid]-dist[i];
	}
	for(register int i=mid+1;i<=r;++i){
		rt+=dist[i]-dist[mid];
	}
	return rt;
}//If the number of villages is odd, put the post office in the median distance and . If the number of villages is even, place any one between the middle two villages.
// so we can enumerate [l,mid] ,(mid,r] ,calc villages' value in left and right range
//because we only put one post office here ,so we can enumerate it from two endpoint to the median distance. 

int main(){
	scanf("%d%d",&n,&m);
	memset(f,0x7f7f7f,sizeof(f));
	f[0][0]=0;//init
	for(register int i=1;i<=n;++i){
		scanf("%d",&dist[i]);
	}
	sort(dist+1,dist+1+n);//sort,because dist wasn't monotonous 
	for(register int i=1;i<=n;++i){ //enumerate Villages
		for(register int j=1;j<=m;++j){ //enumerate post office
			for(register int k=0;k<i;++k){ //enumerate range
				int value=calc(k+1,i); // min dist of range[k+1,i]
				f[i][j]=min(f[i][j],f[k][j-1]/*we put one post office on "k"*/+value);
			}
		}
	}
	printf("%d\n",f[n][m]);
	return 0;
}//This problem focuses on enumerating post office locations, so let's add one more layer of loop enumeration to the post office location from the current location to the first location
